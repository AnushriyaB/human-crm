human-crm
A personal CRM to help you stay close to the people who matter most.

Features
World map — pin friends to locations, with cluster markers for co-located friends
Friend profiles — modular bento-style cards for identity, timeline, favorites, work, social links, notes, and collages
Passkey sharing — invite friends to fill out their own profiles via a shareable passphrase
Tactile design system — neuromorphic UI with a consistent token-based theme (light; dark coming later)
